# ADRs do Projeto Webserv

Este diretório contém os Architecture Decision Records (ADRs) do projeto Webserv.

## Ordem Recomendada de Implementação
1. ADR-001 (Modelo de concorrência e multiplexação I/O)
2. ADR-002 (Sockets e ciclo de vida de conexões)
3. ADR-003 (Parsing HTTP/1.1 e framing)
4. ADR-004 (Parsing e validação de configuração)
5. ADR-005 (Roteamento)
6. ADR-006 (Serviço de arquivos e autoindex)
7. ADR-007 (Tratamento de erros)
8. ADR-008 (Execução de CGI)
9. ADR-009 (Upload multipart)
10. ADR-010 (Memória e limites)
11. ADR-011 (Virtual hosts)
12. ADR-012 (Sessões/cookies – bônus)
13. ADR-013 (Múltiplos CGIs – bônus)

## Referências RFC
